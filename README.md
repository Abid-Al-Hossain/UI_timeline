# UI Foundry Timeline Studio

Standalone, self-contained Timeline component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Content model: item count, labels, metadata, actions, empty/loading states
- Structure: orientation, density, hierarchy, separators, active/selected/expanded state
- Interaction: keyboard semantics where APG defines them, hover/focus/active previews
- Accessibility: roles, labels, heading levels, table/list/tree/tab relationships
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
